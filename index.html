🧠 행동경제학 원칙
손실 회피: "지금 안 바르면 기미가 생깁니다" — 이득보다 손실이 2배 강하게 작용
사회적 증거: 리뷰 수, 임상 참여자 수, "~명이 선택했습니다"
희소성/긴박감: "초도 한정 수량", "오늘까지 할인"
닻내리기: 정가 → 할인가 순서로 가치 지각 상승
프레이밍: "기미 치료"(부정) vs "기미 방어선 구축"(능동/긍정)
💡 설득의 심리학 6원칙 (Cialdini)
1.상호성: 무료 샘플, 정보 제공
2.헌신/일관성: 페인포인트 공감 → 제품
3.사회적 증거: 실제 사용 후기, 임상
4.권위: 성분 특허, 피부과 추천
5.호감: 브랜드 스토리, 황칠 헤리티지
6.희소성: 한정 수량/기간

후킹 섹션
기미가 보이기 시작한다면, 이미 늦었다. — 문제 직면 + 강렬한 비포애프터 이미지
손실 회피 + 첫 인상
02
브랜드 스토리
조선 왕실이 선택한 황칠나무 → 현대 피부 갑옷으로 — K-헤리티지 후킹
호감 + 스토리텔링
03
핵심 성분 3단계
트라베논(멜라닌 억제) + 황칠(항산화 방어) + 에어로젤 피막(성분 고정) = 기미 방어선 3겹
권위 + 원리 이해
04
임상 결과
기미 개선 ~% (인체적용시험 완료) — 수치와 그래프로 시각화
사회적 증거 + 권위
05
제형 차별화
에어로젤 나노 피막 — 공기처럼 가볍지만 성분은 피부에 밀착 고정 — 비주얼 시연
차별화 + 작동 원리
06
실제 후기
실제 구매자 비포애프터 + 텍스트 리뷰 모음
사회적 증거
07
FAQ
자주 묻는 질문 — 자극 없나요? 임신 중 사용? 다른 제품과 병행?
불안 해소
08
구매 전환 CTA
오늘까지 특가 + 무료 배송 + 30일 환불 보장 — 긴박감 + 리스크 제거
